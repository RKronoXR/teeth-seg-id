# CHANGELOG

## v0.1.0 academic demo

### Added

- Training pipeline for panoramic tooth segmentation.
- Panoramic inference CLI.
- Streamlit application for academic demonstration.
- Interactive zoom and pan image viewer.
- FDI tooth selector.
- Selected-tooth review panel.
- Interactive Plotly result charts.
- Detected vs missing FDI visualization.
- Interactive 32-tooth FDI map.
- Clinical correction panel.
- Persisted clinical corrections CSV.
- Automatic findings summary.

### Scope

This version is an academic and portfolio demo for tooth segmentation and FDI identification in panoramic radiographs.
