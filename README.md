# Teeth Segmentation and Identification

Pipeline for tooth instance segmentation and FDI tooth numbering in panoramic dental radiographs.

## Goals

1. Tooth instance segmentation
2. FDI tooth identification
3. Training on NVIDIA DGX Spark
4. Objective evaluation
5. Future ONNX/TensorRT export
6. Academic and potential commercial use

## Main dataset

UFBA-425 will be used as the primary dataset, subject to license verification and proper attribution.
